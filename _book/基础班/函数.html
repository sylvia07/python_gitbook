函数

  • 概念：将具有独立功能的代码块组织成为一个整体，使其具有特殊功能的代码集

  • 作用：加强代码的复用性，提高程序编写的效率

  • 基本语法 ```

    函数定义：形参

    def 函数名(参数)：
    函数体 return 结果 # 根据你的需求来决定是否return

函数调用：实参

变量名 = 函数名(参数)

```
