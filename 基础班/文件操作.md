#文件操作
---
###目标：
	* 会对文件进行读取写入操作
---

####基本语法
```
格式一（手工关闭格式）：
	1.打开文件：	file = open("文件名","读写模式")
	2.操作文件：	*
	3.关闭文件:	file.close()
格式二（免关闭格式）：
	1.打开文件:	with open("文件名","读写模式") as file:
	2.操作文件:	*
	3.关闭文件(自动关闭文件)

```
